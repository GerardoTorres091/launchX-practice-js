# launchX-practice-js

### Using:

- Html5
- Css
- js vanilla
- Bootstrap 5
- nes css styles


### practice:
- make a pokedex
- Have an input input, where the name of the pokemon to search for is entered.
- Take that input and go to consult the pokeapi API
- show on the screen the following data of the entered pokemon:
- Name
- Image
- pokemon type
- statistics
- movements
